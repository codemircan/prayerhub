# 🕌 Global Prayer Times App

A modern, production-ready Global Islamic Prayer Times web application built with React 18, Vite, and TailwindCSS.

## 🚀 Features

- 📍 **Location Detection**: Automatically detects your location for accurate prayer times.
- 🔍 **City Search**: Search for over 50+ major world cities.
- 📅 **Monthly View**: View prayer times for the entire month and export to CSV.
- 🕋 **Qibla Compass**: Calculate the direction and distance to the Kaaba.
- 🌍 **Multi-language**: Supports English, Turkish, and Arabic (with RTL layout).
- 🌓 **Dark Mode**: Toggle between light and dark themes.
- 📱 **PWA Support**: Installable on mobile devices with offline caching.

## 🛠️ Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: TailwindCSS & shadcn/ui
- **State Management**: Zustand
- **API Fetching**: TanStack React Query & Axios
- **Internationalization**: i18next
- **Date Utilities**: date-fns
- **PWA**: vite-plugin-pwa

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

Build for production:
```bash
npm run build
```
